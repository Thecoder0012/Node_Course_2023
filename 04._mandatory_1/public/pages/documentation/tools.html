Tools


In this course we have used npm (Node package manager) to install packages.

To initialise:

        
          npm init
          or
          npm init -y (yes to default config)

To install/uninstall a package you simple type:

            
            npm install express
            npm i express // abbrevation for install
            npm uninstall express // uninstalling package

Nodemon is a tool for the development software. It restarts the node application, when it detects changes in the code. 

          
            npm install nodemon

            example: 
            nodemon app.js

If you want nodemon to detect html files, you can configure it in the nodemon.json file.

          
            {
              "ext": "js, json, html"
            }
