Time/date and fetch


There are several ways to work with time and date in JavaScript.

The most general way to use date and time is to use the Date class and instantiate an object from it.

            
            const date = new Date();
            const year = date.getFullYear();

It is also possible without the new keyword.
Without the new keyword, you get some additional info about the datetime.

            
              const date = Date();
              console.log(date) // Sat Apr 02 2023 14:43:20 GMT+0200 (Central european summer time)

Fetching from apis can be done with the Fetch Api.
Either with async await or with promises 

            Fetch("api/bikes")
                .then(response => response.json())
                .then(result => console.log(result))
