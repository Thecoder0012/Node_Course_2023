Rendering html and loops


To render html files, you need to 'return/send' the files in the get routes.

Instead of using the send method on the response object, you should use the sendFile method.

Before that you need to import path or simply just use __dirname + filepath.

If the html files are in the public folder, then you should tell express to use that static folder.

           
            import express from 'express';
            const app = express();
            import path from 'path';

            app.use(express.static("public")); // express has a static folder named public

            app.get("/", (req, res) => {
              res.sendFile(path.resolve(path.html)); // path.resolve() gives the absolute path for a file
            });

Loops: Map, filter, Find methods

Map is used to create a new array with the existing values of an array that may be modified through the callback function.

Filter is used to filter through an array with a match to the condition

Find returns a specific element through the condition

            
                letters.map((letter) => letter.toUpperCase());
                numbers.filter((number) => number > 5);
                values.find((value) => value.age === age);
