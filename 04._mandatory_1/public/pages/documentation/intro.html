Variables/data types and REST conventions


Variables are defined with these keywords:

            let, var, const.
            
Define varibale and assign it a value:

            const country = "Denmark"

The difference between let and var is about scoping, and that those variables may be reassigned.

However, it is not possible to reassign a const variable.

The most common data types are:

            String, Number, Boolean, Undefined, Null, Object.

RESTful web services is a way to expose apis that contains data about something specific.
The protocol used is http. Apis can be created for anything. For instance cars.
The http requests will then be send to a defined car endpoint. 

            
                get request ("/cars")
                post request ("/cars")
                get request ("/cars/id")
                put request ("/cars/id")
                patch request ("/cars/id")
                delete request ("/cars/id")
