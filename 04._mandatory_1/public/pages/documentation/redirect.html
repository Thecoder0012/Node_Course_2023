Server side rendering


If you want to redirect in the client, you can use the window object. 

            
                window.location = "/home";
                window.location.href = "/home";
                window.location.assign("/home");
                window.location.replace("/home"); // no going back to the Referer page (where the request is sent from)

Redirection in the backend with express can also be done quite easily. 

            
                app.post("/login", (req, res) => {
                const { username, password } = req.body;

                if (username === "User" && Number(password) === 1234) {
                    res.redirect("/homepage");
                } else {
                    res.redirect("/");
                }
            });

In the above code, we see that the redirection is used with the response object -
meaning that the server redirects you based on the if conditions.
