Export/import


There are two ways to export/import in JavaScript. Either by using Commonjs or the type module (ES module).

Example of Commonjs:

            
            module.exports{function}
            const function = require('function.js');

Example of ES module

            
                export function(){}
                import function from 'function.js'

                export default {
                    function1,
                    function2
                }

                import func from 'function.js'
                func.function1();

In a html file it is also possible to specify the module in a script tag if you want to that file to use the ES module: 

        
            script type="module" src="script.js">/script

How to avoid Cross Site Scripting(XSS) in html/javascript. There are a couple of ways to avoid it.

Firstly, you should stop using innerHTML in the DOM. Instead use: innerText or textContent
