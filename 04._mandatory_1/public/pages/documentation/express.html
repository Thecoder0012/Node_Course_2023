Express/functions


Express is a fast web framework for Node JS:

To use express in a Node project, you need to install it with Node Package Manager (npm).

            npm install express

When installed you need to import/require it and then instantiate it. For this documentation, I will use the way of es modules.

Installing it also sends the dependency to the file called package.json which has information about about our project and dependencies.

            
                import express from 'express';
                const app = express();

You can use express to easily create backend routes

Here I will map a http get request to "/" and the callback function be executed.
The callback function takes a request and response object. 

            
                app.get("/", (req, res) => {
                  res.send({message: "Hello World!"});
                });

Lastly, you need to start the server on a specfic port

            
                app.listen(8080, () => {
                  console.log("Server started on port:", 8080);
                });

There are three ways to define a function in JavaScript.

                
                    function sum(a,b){ // function declaration
                        return a + b;
                    }
                    const sum = function(a,b){ // function expression
                        return a + b;
                    }

                    const sum = (a,b) => { // arrow function
                        return a + b;
                    }
